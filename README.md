# Cloudproof C/C++

C/C++ bindings for Cosmian Cloudproof Encryption.

## Examples

* Compile and run C example

```bash
gcc example/main.c -I include/ -L lib/ -lcloudproof
LD_LIBRARY_PATH=lib ./a.out
```

* Compile and run C++ example

```bash
g++ example/main.cpp -I include/ -L lib/ -lcloudproof
LD_LIBRARY_PATH=lib ./a.out
```
